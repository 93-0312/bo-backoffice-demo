# 06. SID 지급 보류 처리

**경로:** 정산 리스트 > 지급보류금 → SID 지급 보류 처리 · 데모 (예정, 기획 기준 문서)

## 화면 개요

특정 셀러(SID)의 정산 지급을 **보류**하고, 보류 대상 거래(STID)를 관리하는 화면입니다.
[03 정산 리스트](./03-settlement-list.md)의 **지급보류금** 컬럼에서 진입합니다.

## 화면 구성

### 지급보류 SID 리스트
- 지급 보류된 SID 목록과 **합계 금액**

### STID 목록 (선택 SID의 보류 거래)
| 컬럼 | 설명 |
|------|------|
| SID | 셀러 식별자 |
| 셀러명 | 셀러명 |
| 분류 | 거래 분류(SALE 등) |
| 결과 | 거래 결과(SUCCESS 등) |
| 통화 | 통화 |
| 금액 | 거래 금액 |
| 정산금액 | 정산 대상 금액 |
| 정산일 | 정산 예정/실행일 |

## 동작

1. 정산 리스트에서 지급보류금이 있는 가맹점의 **지급보류금**을 클릭해 진입.
2. 지급보류 SID와 그에 속한 STID 거래를 확인.
3. 합계 금액으로 총 보류 규모를 파악하고, 보류 해제/지급 처리를 진행.

## 예외 · 유의사항

- **심사 제외 가맹점**: 정산 계좌 정보가 없으면 지급 처리가 막힌다.
  - 안내: *"정산 계좌 정보를 확인할 수 없습니다. 정상적인 서비스 이용을 위해 추후 계좌 정보를 수정해주세요."*
  - 계좌 정보 조회 중에는 "계좌 정보 조회 중..." 상태를 표시.
