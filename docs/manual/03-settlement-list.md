# 03. 정산 리스트

**경로:** 셀러 > 분할정산 > 정산 리스트 · 데모 `/settlement/list`

## 화면 개요

가맹점·셀러별 **정산 현황**을 회차 단위로 조회하는 화면입니다. 기존 정산서(일반) 내
셀러 일반 정산 내역을 함께 보여줍니다.

> **정산방식:** `MID정산`(가맹점 단위) / `셀러별정산`(SID 단위)
> **현행은 `MID정산`만 제공** — 셀러별정산(차액정산)은 차후 추가되며, 그때 selectbox로 선택하게 됩니다.
> (현재는 선택 불가하게 처리)

## 화면 구성

### 필터
- **날짜 범위** (정산일자 기준) — **최대 검색가능 기간: 12개월**
- **정산 상태** — 지급 완료 / 지급 대기 / 지급 보류
- **검색어** — MID · 가맹점명
- **초기화** / **다운로드**

### 목록 컬럼
NO · MID · 가맹점명 · 가맹점 상태 · 셀러 정산유형 · 셀러 수 · 정산 회차 · 정산일자 · 거래통화 · 정산금액 · 정산 통화 · 확정 정산금액 · 정산 상태 · **지급보류금**

- **가맹점 상태**: LIVE / TEST / 해지(TERMINATE) / 정지(SUSPEND)
- **지급보류금**: 지급 보류된 금액(있으면 강조 표시). 보류 처리는 [06 SID 지급 보류](./06-sid-hold.md) 참고.

## 동작

1. 정산일자 기준 최대 12개월 범위로 정산 회차를 조회한다.
2. **정산금액**(예정)과 **확정 정산금액**(확정분)을 함께 보여준다.
3. 정산 상태로 지급 진행 단계를 확인한다.

## 예외 · 유의사항

- 검색 기간 **최대 12개월**.
- 셀러별정산은 현재 비활성(현행 MID정산만). 추후 차액정산 도입 시 활성화 예정.
