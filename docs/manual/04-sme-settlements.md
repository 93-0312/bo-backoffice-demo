# 04. 영중소 정산서 목록

**경로:** 영중소 정산서 목록 · 데모 `/settlement/sme`

## 화면 개요

영세·중소(SME) 가맹점의 **정산서**를 조회하는 화면입니다. 기존 영중소 정산서 내
셀러 영중소 정산을 합쳐 보여주며, [03 정산 리스트](./03-settlement-list.md)와 같은 정산 레코드를
**영중소 관점(그룹ID·영중소 지급보류금)** 으로 표시합니다.

## 영중소 취급 설정 (중요)

영중소 정산서에 잡히려면 가맹점 설정이 다음을 만족해야 합니다.

- 가맹점 정보 > **셀러 탭** > **분할정산 Check** → 분할정산 관련 내용이 아래에 노출
- **셀러 Y + 셀러 영중소 Y** → **영중소 차액정산 리스트** 대상이 됨
- 영중소(Y/N) 플래그 의미:
  - **MID 영중소**: `MID 영중소 Y` + `셀러 영중소 N`
  - **MID에 엮인 SID 영중소**: `MID 영중소 Y` + `셀러 영중소 Y` (이때 MID 자체는 영중소 취급 X)

> 셀러 탭의 영중소(Y/N)는 "지금 영중소다"가 아니라 **"영중소에서 값을 불러오겠다"는 플래그**입니다.

## 화면 구성

### 필터
- **정산지급일** 범위 — **최대 12개월**
- **정산 상태** — 지급 완료 / 지급 대기 / 지급 보류
- **검색어** — MID · 가맹점명 · 그룹ID
- **초기화** / **다운로드**

### 목록 컬럼
NO · MID · 가맹점명 · **그룹ID** · 가맹점 상태 · 셀러 정산유형 · 셀러 수 · 정산 회차 · 정산일자 · 거래통화 · 정산금액 · 정산 통화 · 확정 정산금액 · 정산 상태 · **영중소 지급보류금**

> 정산 리스트와 차이: **그룹ID**, **영중소 지급보류금** 컬럼이 추가됩니다.

## 동작

1. 가맹점이 특정 기간(정산일 기준)의 **총 차액 정산 금액**을 확인할 수 있는 영역을 제공한다.
2. 그룹ID 단위로 영중소 정산서를 묶어 조회한다.

## 예외 · 유의사항

- 셀러 영중소(Y) 설정이 되어 있어야 이 목록에 노출된다.
- MDR(가맹점 수수료) 계산 시 **영중소 MDR 금액**을 별도 반영한다($MDR금액 − 영중소 MDR금액$).
